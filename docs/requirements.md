## 1. システム概要・背景

### 1.1 目的

学習目的で構築する、X（旧Twitter）ライクなテキストベースのSNSアプリケーション。
受講生・個人が複数人で利用することを前提とし、タイムライン・コメント・いいね・フォローなどの基本的なSNS機能を実装する。

### 1.2 差別化ポイント

| 機能 | X（Twitter） | RaiseTimeLine |
|------|-------------|---------------|
| タイムライン表示 | あり | あり |
| コメント | あり | あり |
| いいね | あり | あり |
| 画像投稿 | あり（最大4枚） | あり（最大4枚） |
| フォロー／フォロワー | あり | あり |
| インプレッション表示 | あり | **なし** |
| リツイート | あり | **なし** |
| 引用リツイート | あり | **なし** |
| DM | あり | **なし** |
| 通知 | あり | **なし** |

## 3. 機能概要

### 3.1 機能スコープ（IN）

| カテゴリ | 機能 | 概要 |
|---------|------|------|
| 認証 | ユーザー登録 | メールアドレス・パスワードで新規アカウントを作成する |
| 認証 | ログイン | メールアドレス・パスワードで認証しセッションを開始する |
| 認証 | ログアウト | セッションを終了する |
| タイムライン | 全体タイムライン | 全ユーザーの投稿を時系列（新着順）で表示する |
| タイムライン | フォロー中タイムライン | フォロー中のユーザーの投稿のみを時系列で表示する |
| 投稿 | テキスト投稿 | 最大280文字のテキストを投稿する |
| 投稿 | 画像付き投稿 | 1投稿につき最大4枚の画像を添付して投稿する（AWS S3保存） |
| 投稿 | 投稿編集 | 自分の投稿を編集する |
| 投稿 | 投稿削除 | 自分の投稿を削除する |
| コメント | コメント投稿 | 任意の投稿にコメントを投稿する |
| コメント | コメント編集 | 自分のコメントを編集する |
| コメント | コメント削除 | 自分のコメントを削除する |
| コメント | コメント数表示 | 投稿に対するコメント数を表示する |
| いいね | いいね | 任意の投稿にいいねをする |
| いいね | いいね取り消し | いいねを取り消す |
| いいね | いいね数表示 | 投稿に対するいいね数を表示する |
| フォロー | フォロー | 任意のユーザーをフォローする |
| フォロー | フォロー解除 | フォローを解除する |
| フォロー | フォロワー一覧 | 自分のフォロワー一覧を確認する |
| フォロー | フォロー中一覧 | 自分がフォロー中のユーザー一覧を確認する |
| ユーザー検索 | ユーザー名検索 | ユーザー名（部分一致）でユーザーを検索する |
| ユーザー検索 | 検索結果のフォロー状態表示 | 検索結果にフォロー中／未フォローを表示する |
| プロフィール | プロフィール表示 | アバター画像・ユーザー名・自己紹介・フォロー数・フォロワー数・投稿一覧を表示する |
| プロフィール | プロフィール編集 | 自分のアバター画像・ユーザー名・自己紹介を編集する |

### 3.2 機能スコープ（OUT）

- リツイート・引用リツイート
- インプレッション（閲覧数）表示
- DM（ダイレクトメッセージ）
- 通知機能
- ハッシュタグ検索
- トレンド表示
- ブックマーク
- リスト機能

## 4. 画面概要

## 5. データ概要

主要エンティティは以下の通り。詳細は [データモデル・ER図](data-model.md) を参照。

| エンティティ | 概要 |
|-------------|------|
| users | ユーザー情報（認証情報・プロフィール） |
| posts | 投稿（テキスト・作成日時） |
| post_images | 投稿に添付された画像（S3 URL・表示順） |
| comments | 投稿へのコメント |
| likes | 投稿へのいいね（ユーザーと投稿の紐付け） |
| follows | フォロー関係（フォロワーとフォロー先の紐付け） |
